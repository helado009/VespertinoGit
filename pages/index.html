¿Qué es el DOM?

El DOM (Modelo de Objetos del Documento, por sus siglas en inglés Document Object Model) es una interfaz de programación que representa y permite manipular documentos HTML y XML. A través del DOM, se puede acceder, modificar, agregar o eliminar elementos dentro de un documento, proporcionando una estructura jerárquica en forma de árbol.

Características del DOM

Acceso Dinámico

Permite acceder a los elementos del documento de manera dinámica, lo que posibilita su manipulación en tiempo real.

Estructura Jerárquica

La estructura del DOM se organiza en nodos, donde cada nodo tiene una relación jerárquica.

Manipulación de Contenido

Permite modificar la estructura, el estilo y el contenido de los elementos dentro del documento.

Interacción con JavaScript

A través de JavaScript, el DOM permite actualizar la página y responder a las acciones del usuario.

Objetivos del DOM

Manipulación Eficiente

Facilitar la manipulación eficiente de la estructura de los documentos HTML y XML.

Interactividad Dinámica

Permitir la interacción dinámica entre el contenido de la página y el usuario.

Optimización de Páginas

Optimizar la creación de páginas web interactivas y accesibles.

Estándares de Manipulación

Proveer una forma estandarizada para acceder y manipular la estructura del documento.

El Objeto Document

El objeto document es la base del lado del cliente en las páginas web. Permite acceder y manipular los elementos del documento HTML, como el cuerpo, imágenes, formularios y la cabecera.

Propiedades Comunes

Propiedad Descripción
document.body Devuelve el elemento <body>.
document.title Obtiene o cambia el título de la página.
document.images Lista todas las imágenes de la página.
document.bgcolor Cambia el color de fondo con nombres como "red" o valores hexadecimales como #FFFFFF.

Métodos del Objeto Document

# Método Descripción
1 open() Abre el documento para edición.
2 close() Cierra el documento.
3 write() / writeln() Inserta contenido HTML dinámico.

Navegación en el Árbol DOM

El DOM organiza la página en una estructura jerárquica similar a un árbol. Este árbol está compuesto por nodos, y cada nodo puede tener nodos hijos, padres y hermanos. Al manipular la página, es posible navegar por este árbol para acceder o modificar diferentes elementos.

  • parentNode: Devuelve el nodo padre de un elemento.
  • childNodes: Proporciona todos los nodos hijos de un elemento.
  • firstChild y lastChild: Acceden al primer y al último nodo hijo de un elemento.
  • previousSibling y nextSibling: Permiten recorrer los nodos hermanos de un elemento.

Acceso a los elementos HTML

  • getElementById(id): Encuentra un elemento específico usando su atributo id.
  • getElementsByName(): Obtiene todos los elementos con una etiqueta específica.
  • getElementsByTagName(): Recupera todos los elementos que coinciden con la etiqueta proporcionada.

Cambio de Elementos

Puedes cambiar el contenido o los atributos de un elemento con los siguientes métodos:

  • innerHTML: Cambia el contenido HTML dentro de un elemento.

  • setAttribute(atributo, valor): Cambia el valor de un atributo de un elemento. También se puede cambiar el atributo o el estilo de un elemento utilizando:
        
            elemento.atributo = valor; 
            elemento.style.propiedad = estilo;
